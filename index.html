Primeros Pasos con Docker

Paso 1: Crear un contenedor Docker

Utiliza el siguiente comando para crear un contenedor Docker con volúmenes: 

docker run -v /ruta/local:/ruta/contenedor -d nombre-imagen

Paso 2: Verificar volúmenes

Puedes verificar los volúmenes montados utilizando el siguiente comando: 

docker inspect nombre-contenedor

Paso 3: Persistir datos

Los datos en el volumen /ruta/local en tu máquina host se persistirán incluso si el contenedor se detiene o elimina.
